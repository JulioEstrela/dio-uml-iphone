# UML Iphone
O projeto simula a UML do Iphone 1 em Java, com base na apresentação de Steve Jobs em 2007.

# Aplicativos
## 🎵 Reprodutor Musical - "Ipod"
Funcionalidades:
- Selecionar Música
- Tocar Música
- Pausar Música

## ☎️ Telefone
Funcionalidades:
- Ligar
- Atender
- Iniciar Correio de Voz

## 🛜 Internet
Funcionalidades:
- Adicionar nova aba
- Selecionar aba
- Acessar página
- Atualizar página
